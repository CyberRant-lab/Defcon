# Homelab Setup Guide

## Introduction
This guide provides an overview of setting up a homelab environment.

## Setup Instructions

1. Navigate to the demo directory:
   ```bash
   cd <DEMO_DIR>
   ```
2. Ensure you have the following prerequisites installed:
   - Tool X
   - Tool Y

3. Follow the steps below:
   - Clone the repository:
     ```bash
     git clone <REPO_URL>
     ```
   - Ensure your folder structure resembles:
     ```plaintext
     <DEMO_DIR>/
       ├── Tool X
       └── Tool Y
     ```

4. Execute the demo script:
   ```bash
   bash <DEMO_DIR>/script.sh
   ```
